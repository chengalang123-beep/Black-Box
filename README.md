# StreamBox - Vercel Streaming App

A Netflix-style streaming web app built with React and Vite.

## Features

- Streaming-style landing page
- Built-in HTML5 video player
- Demo playable streams
- Browse categories
- Live TV-style section
- Continue watching progress bars
- My List / watchlist saved in localStorage
- Upload and preview your own local video file in the browser
- Fully responsive design
- Ready for Vercel deployment

## Important note

This is a front-end streaming app template. The upload feature previews videos locally in the browser only. For real user accounts, paid subscriptions, cloud video hosting, or permanent uploads, connect a backend service like Firebase, Supabase, Mux, Cloudflare Stream, or AWS S3.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
